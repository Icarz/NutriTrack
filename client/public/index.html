CSS custom properties

Drop this into your global stylesheet (e.g. :root in app.css) and reference with var(--color-accent).

:root {
  --color-ink:        #0F2A22;
  --color-accent:     #2E8B5F;
  --color-accent-soft:#7DBA8A;
  --color-paper:      #F4F1EA;
  --color-paper-warm: #FAF8F2;
  --color-stone:      #6B7066;
  --color-rule:       #DDD7CB;

  --font-display: 'Manrope', system-ui, sans-serif;
  --font-body:    'Inter',   system-ui, sans-serif;
}

Do

✓ Use the primary lockup on light surfaces.
✓ Use the dark variant on #0F2A22 or imagery.
✓ Scale uniformly — the SVG has no fixed pixel size.
✓ Use currentColor mono inside icon buttons.

Don't

✗ Recolor the leaf to anything outside the green family.
✗ Stretch, rotate, or outline the wordmark.
✗ Place the primary green wordmark on a low-contrast background.
✗ Embed the pulse line as a separate element from the wordmark.
